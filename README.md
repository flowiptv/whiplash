# whiplash
A library for displaying TV-friendly GUI.

Supports air mice, remotes, gamepads, and keyboards. (Keyboard input does NOT work on the on-screen keyboard)

Notice: some assets should be in /opt/flowtv.
Such as `/opt/flowtv/whiplash.json`, `/opt/flowtv/font.ttf`, `/opt/flowtv/pointer.png`.
This does also mean the `/opt/flowtv/` directory must exist.

## requirements
- pygame
- python3
